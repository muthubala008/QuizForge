# QuizForge — AI Quiz Generators 

Turn your notes into a quiz instantly using AI!

## Live Demo
link: https://muthubala008.github.io/QuizForge

## Features
-  Upload PDF, Image, or Text files
-  AI generates 10 quiz questions automatically
-  Timer for each question
-  Score screen with breakdown
-  Generate next 10 different questions
-  Dark / Light mode

## Built With
- HTML, CSS, JavaScript
- Google Gemini AI API
